# Research Archives

This directory serves as the repository for literature, research notes, and historical report snapshots for the EVolvAI project.

## Index

- **`ResearchReport.md`**: Theoretical foundations of the LinDistFlow physics engine and the VAE-GA architectural coupling.
- **`DATA.md`**: Documentation for the IEEE Dataport and ACN data sources used in early project phases.
- **`Experiments.md`**: Detailed configuration for the NYC-pivoted experimental runs.
- **`Results.md`**: Performance tracking for model loss, reconstruction fidelity, and constraint adherence.
- **`ROADMAP.md`**: High-level trajectory and integration milestones.
- **`Resources.md`**: Aggregated notes on grid specifications, transformer limits, and external NYC data schemas.
- **`PaperCheckpoints/`**: Contains PDF versions of paper drafts and presentation slides.
- **`EVolvAI_Training_Archive.ipynb`**: Previous iterations of the training pipeline (pre-NYC pivot).
